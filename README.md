## 🔧 설치 및 실행

### 1. 의존성 설치
```bash
npm install
cd functions && npm install
```

### 2. 로컬 개발 서버 실행
```bash
firebase emulators:start --only functions
```

### 3. 프로덕션 배포
```bash
firebase deploy
```
